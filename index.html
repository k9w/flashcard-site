An operator operates on one or more operands.

A variable stores data that can optionally be changed.

A good Variable Naming Convention in JavaScript is lowerCamelCase, where the variable name is one or more words describing what it does. The first word is always lower case. Any second and subsequent words in the variable name are capitalized (just the first letter uppercased). All words are right together without any white space or other separators. For example: "myTestVariable".

A function is a snippet of code statements which can be called from other statements in the code.

A method contains a function and can be used on an object.

An argument is the value supplied to a variable or parameter.

A parameter is a variable of input in a function.

Return sends a value from the function, code block, or program file to the function/block/file that originally called it.
