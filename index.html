Flashcards for JavaScript and jQuery


Programming language for dynamic web content supported by most major browsers.


An operator operates on one or more operands.


A variable stores data that can optionally be changed.


A good Variable Naming Convention in JavaScript is lowerCamelCase, where the variable name is one or more words describing what it does. The first word is always lower case. Any second and subsequent words in the variable name are capitalized (just the first letter uppercased). All words are right together without any white space or other separators. For example: "myTestVariable".


A function is a snippet of code statements which can be called from other statements in the code.


A method contains a function and can be used on an object.


An argument is the value supplied to a variable or parameter.


A parameter is a variable of input in a function.


Return sends a value from the function, code block, or program file to the function/block/file that originally called it.


Chaining lets us run multiple methods on something in the same statement.


A string in JavaScript holds a set of characters in quotation marks.


A boolean holds a value of 'true' or 'false'.


A variable, function, or other primative usually is undefined if it is called or referenced without first being declared.


NaN means Not A Number.


Unlike 'undefined', 'null' is an assigned value and means 'nothing'.


alert() is a built-in JavaScript function that pops up a message on the screen which must be dismissed by clicking 'OK'.


A comment is not evaluated or executed in the code. It is a place for the developer to leave a note for another developer or for themselves in the future about what the code is supposed to do.


jQuery is a library of JavaScript functions, methods, etc, available for use in dynamic websites..


An attribute is a bucket of data in a function or object.
